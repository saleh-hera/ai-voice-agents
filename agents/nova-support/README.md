## What Nova Support does
A voiced AI customer service agent that handles:
- 🛠️ **Issue resolution** — order problems, damaged items, missing packages
- 💸 **Refunds and account help**
- 😟 **Complaint handling** — always leading with empathy
- ↗️ **Smart escalation** — passes complex cases to humans with full context

Nova resolves most requests instantly, improving satisfaction while reducing handle time for the team.

## The problem she solves
Support teams drown in repetitive tickets. Nova clears the routine ones 24/7 so humans focus on the hard cases.
